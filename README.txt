World Cup Tracker - GitHub Pages

Files:
- index.html: the website. You usually do not need to edit this.
- scores.js: the nightly score update file. Edit this one.

How to update scores:
1. Open scores.js in GitHub.
2. Click the pencil/edit icon.
3. Add new completed matches to window.SCORE_UPDATES.
4. Commit changes.
5. Refresh your GitHub Pages site.

Example score line:
{ date: 24, team1: 'Suiza', team2: 'Canadá', score1: 2, score2: 1 },

Keep the comma at the end of each line except it is also fine on the last item.
